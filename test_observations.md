# Observações do Teste Visual - To-Do v2

## Funcionalidades Implementadas

### 1. Separação de Visibilidade ✅
- Admin não vê tarefas criadas por lojas (comunicação loja↔gestor)
- Gestor vê apenas tarefas das suas lojas ou criadas/atribuídas a ele

### 2. Tabs Recebidas/Enviadas ✅
- Tab "Recebidas" para tarefas destinadas ao utilizador
- Tab "Enviadas" para tarefas criadas pelo utilizador
- Contador em cada tab

### 3. Destaque Visual "Para Mim" ✅
- Cards com borda azul à esquerda para tarefas recebidas
- Badge "Para si" em tarefas não vistas
- Badge "Visto" após visualização

### 4. Animação Pulse ✅
- Animação suave em tarefas não vistas
- Ring azul para destacar novos itens

### 5. Botão FAB ✅
- Botão flutuante no canto inferior direito
- Abre modal de criação rápida

## Notas
- Testado com utilizador admin (Marco Amorim)
- 4 tarefas na tab "Enviadas"
- Tab "Recebidas" vazia (normal para admin)
